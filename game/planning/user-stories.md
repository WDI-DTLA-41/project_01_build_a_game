# User Stories

## MVP

* Basic functionality
  * Scoring
  * Hit
  * Stay
  * Keep track of $$$
  * Display current hand total
* 1 deck / shuffle after every round
* 1 player vs dealer
* Basic payout 1:1


## Features

* More players
* Eight decks
  * Keep track / discard cards that are dealt
* Shuffle after 50% of cards are dealt
* Split
* Double down
* Insurance
* Soft vs Hard hands
* Table max
* Double bet
* Replay same bet
* Different payout for natural 'Blackjack' on deal (3:2)
